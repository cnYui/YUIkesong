# stitch_flutter

基于 Flutter 实现的时尚穿搭应用，完整复刻多页面 UI 与交互逻辑，涵盖首页、衣柜管理、AI 试穿、个人中心、社区等功能，并参考既有 HTML 设计稿实现一致的视觉效果。

## 功能概览

- **底部导航栏**：提供首页、我的衣柜、社区、AI 试穿室、我的五大入口，支持跨页面状态保持。
- **首页**：展示今日天气、穿搭推荐，支持「立即试穿」快速跳转至 AI 试穿室。
- **我的衣柜**：
  - 固定分类菜单、搜索及筛选入口。
  - 支持多选衣物、浮动悬浮窗操作、相机/相册添加衣物流程。
  - 悬浮窗可快速进入 AI 试穿室。
- **AI 试穿室**：
  - 生成图片/视频模式切换，底部操作按钮随模式变化。
  - 支持保存穿搭到「我保存的穿搭」页面。
  - 底部 PageView 展示生成结果，支持 Indicator 指示。
- **我保存的穿搭**：
  - 显示用户已保存的穿搭组合。
  - 支持卡片多选，选中后显示悬浮操作栏，可删除或发布到社区（示例提示）。
- **社区页面**：
  - 参考设计稿实现瀑布流布局，右列相对左列下移 10%。
  - 顶部标题居中显示。
- **我的页面**：
  - 展示用户信息、快捷入口。
- **子页面**：包括管理自拍、我的收藏、设置、关于我们等。

## 技术要点

- Flutter 3.x + Material Design，自定义 `StitchTheme` 保持应用视觉统一。
- 使用 `ValueNotifier` 管理状态（衣柜选择、保存穿搭等）。
- 大量使用 `CustomScrollView`、`Sliver`、`PageView`、`AnimatedContainer` 等提升交互体验。
- 图片选择采用 `image_picker`，模拟添加衣物的处理流程。
- 悬浮窗、选中态等使用 `Stack`、`Positioned` 构建，贴合设计稿。

## 目录结构

- `lib/main.dart`：应用入口，集中处理底部导航与路由。
- `lib/pages/`：存放各业务页面（home、wardrobe、community、ai_fitting_room、profile 等）。
- `lib/widgets/`：通用组件，如自定义底部导航栏。
- `lib/state/`：基于 `ValueNotifier` 的轻量状态管理。
- `lib/theme/`：主题与颜色定义。
- `docs/`：`frontend_overview.md` 提供前端实现与后端对接说明。
- `stitch_/`：原始 HTML 与设计稿参考资源。

## 快速开始

```bash
flutter pub get
flutter run -d chrome
```

> 运行前请确保已在 Windows 启用开发者模式，以支持插件软链接。

## 注意事项

- 社区、发布等操作目前为前端示例逻辑，后续可接入实际后端接口。
- 测试环境中需注意网络图片加载失败的问题，可替换为本地资源。
- 所有页面均以 HTML 设计稿为准，后续调整请同步更新对应页面代码。
